# cli-1742

前端脚手架工具。

## 安装方法

```bash
npm install cli-1742 -g
```

## 模块介绍

### vue

``` bash
# 1.初始化项目

# - new folder
cli-1742 init-vue new-folder

# - exist folder
cli-1742 init-vue

# 2.安装依赖
npm install

# 3.运行
npm run dev

# 4.打包
npm run build
```
